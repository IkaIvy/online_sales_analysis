# online\_sales\_analysis



\# Online Sales Analysis



\## Project Description

This project is a Python-based system for managing products and analyzing sales in an online store. It demonstrates object-oriented programming and Git version control, including branching, merging and conflict resolution.



\## Features

\- Add, display, update and remove products

\- Manage customer cart

\- Calculate total inventory value

\- Calculate total price for checkout



\## Classes



\### Product

Represents a product with:

\- name

\- price

\- quantity



Methods:

\- display\_info()

\- update\_quantity()



\### ProductManager

Manages product list:

\- add\_product()

\- display\_all\_products()

\- get\_total\_inventory\_value()

\- remove\_product\_by\_name()



\### Cart

Handles shopping cart:

\- add\_to\_cart()

\- display\_cart()

\- calculate\_total()
